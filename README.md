#  MedTrust Frontend

MedTrust is a secure and user-friendly web application designed to provide healthcare professionals and patients with direct access to electronic health records (EHRs).  
This repository contains the **frontend** implementation, developed using **React**, **TypeScript**, and **Mantine UI**, following principles of accessibility, modularity, and responsive design.
